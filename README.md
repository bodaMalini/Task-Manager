# Task-Manager
Built a full-stack Task Manager web application. Implemented features like task creation, priority management, due dates, filtering, sorting, search, dark mode, and important task marking.

## 🚀 Live Demo
https://chimerical-conkies-1c3b8f.netlify.app

## ✨ Features
- Add, delete, and update tasks
- Set priority (High, Medium, Low)
- Due date management
- Mark tasks as completed/undo
- Search and filter tasks
- Sort tasks
- Mark important tasks ⭐
- Dark mode 🌙
  
## 🛠 Tech Stack
- MongoDB Atlas
- Express.js
- React.js
- Node.js
- Netlify (Frontend)
- Render (Backend)
